# フロントエンド仕様 (Next.js)

## 1. UI/UX設計 (レスポンシブ)
- **デスクトップ (3ペイン)**: サイドバー（ハンバーガーメニュー内包）/ 一覧 / 詳細。
- **モバイル (1ペイン)**: リスト形式、タップで詳細へ。
- **マルチアドレス管理**: アカウント切り替えなしで、サイドバーのタブから受信用アドレス（info@, support@ 等）を即座に切り替え。

## 2. 機能
- **既読・未読操作**: ユーザーが任意に「未読」に差し戻し可能。
- **スレッドタイムライン**: 同じUUIDを持つ送受信メールを統合して表示。
- **転送・返信支援**: 
    - 転送ボタン押下時に既存コードを継承、または新規コードを発行して本文挿入。
- **Discord通知連携**: 受信時にDiscordへメンション付き通知（Lambda経由）。

## 3. 認証
- Firebase Auth による認証。APIリクエスト時はヘッダーに ID Token を付与。
